Maggiori dettagli sulle tecnologie frontend

Visual Studio Code :

Visual studio code è un editor al quale può essere attaccata la quantita' di plugin necessaria allo sviluppo di un progetto trasfromandolo nell'atto pratico in un ide, la sua forza è l'elasticità di poterlo usare per sviluppare una ampissima varietà di tipi di progetto mantenendo sempre l'editor come base.

Angular :

Angualar è un framwork improntato prettamente a sviluppare Single Page Applications nella parte frontend di una applicazione, la sua utilità consiste nel modularizzare all'estremo il codice frontend per lo sviluppatore rendendolo più leggibile e veloce da sviluppare, nel pratico introduce il concetto di componente che va a sostituire le directive ed è suddiviso in 4 tipi di files: typescript, spec typescript, html e css. Il componente è una rappresentazione di una entità in una pagina, è contenuto all'interno di un modulo, i moduli hanno lo scopo di modularizzare l'utilizzo dei componenti e di ottimizzarne favorendo il riutilizzo del codice, angular carica i componenti partendo dai moduli. l'ultima parte fondamentale di angular dopo il componente e il modulo sono i servizi, suddetti si utilizzano per funzioni che vengono ripetute all'interno di diversi componenti e hanno la peculiarità di essere vivi durante tutto l'utilizzo dell'applicazione.

GitBash :

GitBash è un sistema di versioning che offre repositories accessibili tramite terminale, il team si è coordinato dividendosi il lavoro appoggiandosi a una repository.

Ng Bootstrap :

Bootstrap è una raccolta di strumenti liberi per la creazione di siti e applicazioni per il Web. Essa contiene modelli di progettazione basati su HTML e CSS, sia per la tipografia,che per le varie componenti dell'interfaccia, come moduli, pulsanti e navigazione, così come alcune estensioni opzionali di JavaScript.

Json-server :

Json server è una REST API simulata, nell'applicazione è stata utilizzata per simulare il backend e consentire al team di utilizzare il pattern di sviluppo Agile, grazie a json server gli sviluppatori frontend e backend hanno potuto lavorare contemporaneamente.
